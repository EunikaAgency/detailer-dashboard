# Products API Backups

This folder stores git-friendly daily snapshots of the live `GET /api/products` response.

Structure:
- `latest.json`: latest successful snapshot
- `latest.txt`: relative path to the latest dated snapshot
- `YYYY/YYYY-MM-DD.json`: dated daily snapshots

The payload includes:
- backup metadata
- SHA-256 checksum of the raw API response
- the parsed API response body

Generated by:
- [products-api-backup-cron.js](/var/public/otsukadetailer/detailer/web/scripts/products-api-backup-cron.js)
